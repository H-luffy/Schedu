# Schedu - 学生课程表管理系统

一个功能强大的学生课程表管理系统，支持多种课表模板，让您的课程表更加个性化。

## 主要功能

- 📊 **多种课表模板**：提供简约、多彩、专业、创意和暗色等多种风格的课表模板
- 📁 **Excel导入**：支持上传.xlsx或.xls格式的Excel文件，自动解析课程信息
- 📷 **图片识别**：支持上传课表图片，通过OCR技术自动识别课程信息
- 🎨 **个性化定制**：选择不同的模板风格，打造专属课程表
- ⚠️ **冲突检测**：自动检测课程时间冲突并高亮显示
- 💾 **数据持久化**：使用localStorage保存课程数据和模板选择
- 📤 **导出功能**：支持将课表导出为Excel文件或图片

## 使用说明

1. 点击"模板风格"按钮，选择您喜欢的课表模板
2. 上传Excel文件或课表图片，系统会自动解析课程信息
3. 查看生成的课程表，系统会自动标记冲突课程
4. 可随时切换模板风格，预览不同效果
5. 点击"导出课表Excel"或"导出课表图片"保存您的课程表

## 技术栈

- Next.js 14 + React 18
- TypeScript
- Tailwind CSS
- Tesseract.js (OCR识别)
- XLSX (Excel处理)
- html2canvas (图片导出)
