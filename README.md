# Schedu - 学生课程表管理系统

一个功能强大的学生课程表管理系统，支持多种课表模板，让您的课程表更加个性化。

## 主要功能

- 📊 **模板库系统**：提供丰富的课表模板库，支持浏览、预览和收藏模板
- 🎨 **自定义模板**：创建和编辑个性化课表模板，支持颜色、圆角、间距等自定义
- 📥 **模板导入导出**：支持导入和导出模板JSON文件，方便分享和备份
- 📁 **Excel导入**：支持上传.xlsx或.xls格式的Excel文件，自动解析课程信息
- 📷 **图片识别**：支持上传课表图片，通过OCR技术自动识别课程信息
- ⚠️ **冲突检测**：自动检测课程时间冲突并高亮显示
- 💾 **数据持久化**：使用localStorage保存课程数据和模板选择
- 📤 **导出功能**：支持将课表导出为Excel文件或图片

## 使用说明

### 基础使用
1. 点击"模板风格"按钮，选择您喜欢的课表模板
2. 上传Excel文件或课表图片，系统会自动解析课程信息
3. 查看生成的课程表，系统会自动标记冲突课程
4. 可随时切换模板风格，预览不同效果
5. 点击"导出课表Excel"或"导出课表图片"保存您的课程表

### 模板库使用
1. 点击首页的"查看更多模板"按钮进入模板库
2. 浏览所有可用模板，点击模板卡片查看预览效果
3. 使用"收藏"按钮保存喜欢的模板
4. 点击"下载模板"可以下载模板JSON文件

### 创建自定义模板
1. 在模板库页面点击"创建自定义模板"
2. 在模板编辑器中调整各项参数：
   - 基本信息：模板名称和描述
   - 颜色设置：背景色、表头、网格线等
   - 样式设置：边框圆角和间距
3. 右侧实时预览效果
4. 点击"保存模板"保存到本地
5. 点击"下载模板"导出为JSON文件

### 导入自定义模板
1. 在模板库页面点击"导入模板"
2. 选择之前导出的JSON模板文件
3. 系统会自动加载并应用该模板

## 技术栈

- Next.js 14 + React 18
- TypeScript
- Tailwind CSS
- Tesseract.js (OCR识别)
- XLSX (Excel处理)
- html2canvas (图片导出)
